# psykopattestet.se (Next.js + Vercel) – MAXXED

SEO-optimerad guide-sajt som leder trafik vidare till ditt test på **relationsvarning.se**.

## 1) Ändra testlänken (om behövs)
Öppna `lib/site.ts`:
- `testUrl` → din riktiga test-URL (med/utan www exakt)

## 2) Köra lokalt (valfritt)
```bash
npm install
npm run dev
```

## 3) Deploy (GitHub + Vercel)
- Skapa nytt GitHub-repo
- Ladda upp filerna (drag & drop) eller via git
- Vercel: Add New Project → importera repo → Deploy

## 4) SEO
- sitemap: `/sitemap.xml`
- robots: `/robots.txt`
- JSON-LD: Breadcrumb + Article på varje sida
- FAQ-schema på vissa sidor

## 5) Extra long-tail sidor
Gaslighting, skuldbeläggning, kontrollbeteende, tystnad som straff, love bombing, traumabindning, gränssättning, isolering, hot/ultimatum.
